🐍 Python Mini Projects Collection

A collection of beginner-friendly Python programs created while learning Python fundamentals such as loops, functions, recursion, lists, string manipulation, and user interaction.

📂 Projects Included
1. 🧠 Quiz Game

A simple command-line quiz game that:

Randomly shuffles questions
Accepts user answers
Calculates the score
Tracks completion time
Allows the user to retry until all questions are answered correctly
Concepts Used
Lists and Dictionaries
Loops
Conditional Statements
random Module
time Module
User Input

A small program that checks whether a fruit is available in a predefined fruit list.

Features
Case-insensitive search
User-friendly output
Uses loop and comparison operations
Concepts Used
Lists
Loops
String Methods
break Statement
for-else Loop
3. 🔢 Factorial Calculator

A recursive Python program to calculate the factorial of a number.

Features
Uses recursion
Handles base case correctly
Displays factorial result
Concepts Used
Functions
Recursion
User Input
Return Values
4. 🔄 Palindrome Checker

A program that checks whether a given string is a palindrome.

Features

Instant result display
Concepts Used
Strings
String Slicing
Conditional Statements

A calculator that computes:

Concepts Used
Mathematical Formulas
